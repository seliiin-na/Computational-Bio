## Python list

#### Q1: turn a list into a list of constituent symbols
```python
myList = list("hello")
```
myList = `['h', 'e', 'l', 'l', 'o']`

#### Q2: randomly shuffle elements of a list
```python
random.shuffle(myList)   # <-- myList will be changed now!
```
myList = `['o', 'l', 'h', 'e', 'l']`


#### Q3: sort a list
```python
myList.sort()
return myList
```
